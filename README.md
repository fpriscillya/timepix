# TIMEPIX@school Pre-lab Orientation

Interactive lesson and digital worksheets for secondary school students
using the MiniPIX radiation detector (CERN Timepix technology).

## Contents

- `index.html` — pre-lab lesson (theory, detector guide, activity previews)
- `activity-1.html` to `activity-5.html` — digital worksheets, fill and print
- `css/` — shared styles
- `js/` — lesson interactions
- `assets/` — images and placeholders

## Usage

Open `index.html` in any browser. No server or build step needed.
Students fill worksheets digitally and print, or use paper version (docx).

## Status

Work in progress. Activities 1 and 2 complete. Activities 3 to 5 drafted.
